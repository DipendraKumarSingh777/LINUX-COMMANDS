50 Common Linux Commands

  • ls – List directory contents.
  • cd – Change directory.
  • pwd – Print working directory.
  • cp – Copy files or directories.
  • mv – Move or rename files or directories.
  • rm – Remove files or directories.
  • mkdir – Create a new directory.
  • rmdir – Remove an empty directory.
  • touch – Create an empty file.
  • cat – Concatenate and display file content.
  • nano – Open a text editor.
  • vi – Open the vi editor.
  • chmod – Change file permissions.
  • chown – Change file owner.
  • df – Display disk space usage.
  • du – Display file and directory disk usage.
  • free – Display memory usage.
  • ps – Show running processes.
  • top – Display system tasks in real-time.
  • kill – Terminate processes.
  • wget – Download files from the web.
  • curl – Transfer data from or to a server.
  • tar – Archive files.
  • gzip – Compress files.
  • unzip – Extract compressed files.
  • find – Search for files in a directory.
  • grep – Search for patterns in files.
  • man – Display command manual.
  • echo – Display message or text.
  • history – Display command history.
  • sudo – Execute command as superuser.
  • useradd – Add a new user.
  • passwd – Change a user’s password.
  • ifconfig – Configure network interfaces.
  • ping – Test network connectivity.
  • scp – Secure copy over SSH.
  • ssh – Secure Shell to a remote machine.
  • iptables – Configure firewall rules.
  • systemctl – Control system services.
  • reboot – Restart the system.
  • shutdown – Power off or reboot the machine.
  • mount – Mount a file system.
  • umount – Unmount a file system.
  • alias – Create shortcut for commands.
  • uname – Display system information.
  • whoami – Display current logged-in user.
  • uptime – Show system uptime.
  • df – Show disk space usage.
  • apt-get – Package manager for Debian/Ubuntu.
  • yum – Package manager for RedHat/CentOS.
